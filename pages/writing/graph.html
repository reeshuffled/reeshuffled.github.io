How to use the graph

Each node is a post. Edges show how posts relate — blue directed edges are explicit links, yellow edges connect semantically similar posts.

Nodes
ClickSelect a post and open the sidebar
Double-clickCenter and zoom in to fit the post and its neighbors
HoverHighlight connected nodes and show labels for the neighborhood
DragReposition a node; the simulation adjusts around it
Navigation
Scroll / pinchZoom in and out
Drag backgroundPan the view
+ / −Zoom in or out in fixed steps
Reset view / RFit all nodes into view (R only works when search is not focused)
W A S D / ↑ ← ↓ →Pan the view (when search is not focused)
EscDeselect the current node and close the sidebar
Sidebar
Linked FromPosts that explicitly link to this one
Links ToPosts this one explicitly links to
Semantically similarPosts with similar topics detected by embedding similarity
Neighbor buttonClick any listed post to jump to it in the graph
Controls
BacklinksToggle directed backlink edges on or off
Semantically similarToggle semantic similarity edges on or off
SearchFilter and highlight nodes by post title
